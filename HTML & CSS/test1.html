1) HTML Skeleton + Comments

Theory 10–12 min Hands-on 15–20 min
Learning goals:
  • Understand HTML document structure: <html> <head> <body>
  • Use headings and paragraphs
  • Write comments in HTML
Experiment steps:
  • Create index.html
  • Edit the code below and click Run
Try: change title, name, and headings.
Output Preview Live
Explain: Title shows on browser tab; comments are for developers; <br> creates line break.

2) Text Formatting + Character Entities

Theory 10 min Hands-on 15–20 min
Learning goals:
  • Use formatting tags: <b>, <i>, <u>, <mark>, <small>
  • Use entities like &copy;, &lt;, &gt;, &amp;
Try: add <strong> and <em>.
Output PreviewLive

3) Hyperlinks + Navigation (Multi-Page Concept)

Theory 10–12 min Hands-on 20 min
Learning goals:
  • Create links with <a>
  • Open link in new tab using target="_blank"
  • Understand site navigation structure
In real folder: make about.html, contact.html and link them.
Output PreviewLive
Class activity: Ask students to create three files and connect using href: index.html, about.html, contact.html.

Mini Project (60–90 min): Student Portfolio / Department Club Page

Assessment Submission
Must include:
  • Header + navigation (3 links)
  • At least 2 cards/articles
  • 1 table (schedule/events/marks)
  • 1 form (registration/contact) with required
  • External CSS OR Bootstrap grid
Quick Viva (5 questions):
  1. What is difference between class and id?
  2. Explain padding vs margin.
  3. What is the use of alt attribute?
  4. What is semantic HTML?
  5. Why viewport meta tag is important?

Optional Extensions (Add 1–2 hours)

  • Extension A: Add a gallery (6 images) + hover effects (:hover)
  • Extension B: Create Bootstrap navbar + footer; keep consistent across pages
  • Extension C: Style forms (focus, spacing, error message area)
  • Extension D: Make a “card grid” using CSS flexbox (optional modern topic)
