5. CSS Position

  • Position defines how an element is placed
  • Relative positions element relative to itself
  • Absolute positions relative to nearest positioned ancestor

  • Flexbox provides one-dimensional layout
  • Aligns items horizontally or vertically
  • Simplifies responsive design

7. CSS Grid

  • Grid provides two-dimensional layout
  • Rows and columns defined together
  • Ideal for page layouts

  • Transitions animate property changes
